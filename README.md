# Бот для Подслушано

## Возможности
- Автоматическая публикация постов из предложок
- Ответы в личные сообщения (статистика, удаление постов, поддержка)
- Защита от спама
- Анонимность по ключевым словам

## Установка

1. Клонировать репозиторий
2. Создать файл `.env` по примеру `.env.example`
3. Установить зависимости: `pip install -r requirements.txt`
4. Запустить: `python main.py`

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| USER_TOKEN | Токен пользователя (для публикации) |
| GROUP_TOKEN | Токен сообщества (для ЛС) |
| GROUP_ID | ID сообщества |
| ADMIN_ID | ID администратора |
| CONFIRMATION_CODE | Код подтверждения Callback API |

## Деплой на Bothost

1. Включить домен
2. Добавить переменные окружения
3. Настроить Callback API в VK
